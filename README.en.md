# SpringBoot-Vue-033_物流管理系统

#### Description
基于SpringBoot + MyBatis Plus + Shiro的物流管理系统，包含管理员与用户两种角色。管理员负责用户、车辆、公告、司机、物流及运单等信息的管理；用户可查看公告与物流信息，并添加运单。采用MySQL数据库存储数据，集成权限控制与工具类实现高效开发。
